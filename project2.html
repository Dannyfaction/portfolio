Description

This was a fun project which we were really excited for. We had to make a 3D game where the screen output had to be different than a normal screen. We chose the Oculus Rift as we had never used it before and wanted to try it out. We also used the Xbox's Kinect Movement tracker.

I was responsible for making the Oculus work and making the Kinect sensor track the swing. I was also responsible for rotating the world smoothly

The Idea of the project was to create a very immersive experience. We used an actual swing which you had to swing on while you had the Oculus on. This created an epic effect where you had the idea you were swinging in outer space.

Our School was so proud of our project's idea that they had signed us up for the Dutch Game Awards. We then showcased our game with the swing to the Jury's which were very happy to test it out. We gained a lot of positive feedback about it. We did however not get nominated because the contenders had games which were made in a timeframe of a year, while we had only 4 weeks to completely make our simulation.

This was my first project with the Oculus Rift and the XBOX Kinect, which was very nice for my learning experience.
