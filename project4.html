Description

My first 2D project in Unity. We had a group of 5 people which consisted out of 2 Developers and 3 Artists. In 8 weeks we had to make a fully working 2D Tower Defense game with different towers.

The other Developer I was working with had little to no knowledge of programming so I had to program everything on my own. Which I had no problems with because that only ment that I was going to learn even more from this project.

We had some problems with spritesheets and there were some bugs in the end, however, we were very proud of the final result.

What I learned in this project was how and when to use animations in Unity, how to use pathfinding with the use of waypoints and also how to make a 'shop' with working User interface.
