Description

This project was my final exam to graduate at Mediacollege Amsterdam. This is the most polished game I have made thus far. The game has a lot of features, functional and technical wise.

The notable functional features are Virtual Reality (cardboard) support, native mobile support and three different ways of movement input.

The notable technical features which improved performance are Object Pooling and GPU Instancing (script wise).

This game was made for a client in collaboration with Mediacollege Amsterdam and the full development (including documentation) took us a total of eigth weeks.

Responsibilities

My main focus this project was making core systems. I was responsible for making the level spawn and also making a tool in the Unity Editor so our artists could easily implement their work in the game.

After completing these systems, I was responsible for improving the overal performance of our game. I put a lot of effort into making an Object Pool and GPU Instancing system which increased performance on Mobile immensely
