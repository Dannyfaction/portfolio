Description

Asteroids was my very first group project. We had learned Actionscript 3 for 2 months prior to this project and we were ready to be working with artists to make a complete game.

The assignment was to make an Asteroids-like game. We gave our own twist to it and made it into an educational game where you needed to solve easy calculations to gain score. The goal was to make learning fun.

The other developer I was working with, was not very fond of Actionscript 3 and had almost no motivation to try to learn the language, which resulted in me having to code everything in the game.

I learned a lot from this project as this was my first. Communication was key for succes. We were very happy with the end result and presented it in front of 50 people. We also learned to use the SCRUM method this project which helped.
